
# Sai - The Next-Generation AI-Native Cloud IDE

Sai is a futuristic, multimodal, AI-augmented development studio designed to unify coding, cloud deployment, security, and collaboration. It allows developers to generate infrastructure, enforce policies, and deploy apps seamlessly across cloud providers using AI.

**Code. Collaborate. Create.**

## 🚀 Key Features

*   **Universal Editor**: Support for TypeScript, Python, Go, Rust, Java, and C/C++.
*   **Sai Co-Pilot**: Integrated chat supporting Text, Image, Video, and Audio generation via Sai Engines (Gemini, Claude, OpenAI).
*   **Sai Terminal**: A robust in-browser terminal supporting `npm`, `pip`, `cargo`, `go`, `docker` and `git` commands.
*   **Governance & Security**: Real-time policy enforcement and compliance scanning (SOC2, HIPAA).
*   **Cloud & DevOps**: One-click Terraform generation, Docker container management, and multi-cloud deployment (AWS, Azure, GCP).
*   **Live Collaboration**: Real-time video calling, screen sharing, and code editing in Sai.

## 🤖 AI Configuration

Sai uses a unified `geminiService.ts` to handle AI requests. It supports:

*   **Sai Core**: Gemini 3 Pro, Gemini 2.5 Flash.
*   **Creative Engines**: Veo (Video), Imagen (Image).
*   **External Links**: Claude 3.5 Sonnet.

To configure API keys, open the **Settings** panel or the **Neural Registry** modal within the application.
